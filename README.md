# Jenkins AWS CI/CD Pipeline

## Project Overview

This project demonstrates an end-to-end CI/CD pipeline using Jenkins, GitHub Webhooks, AWS EC2, Linux, and Python.

The pipeline automatically triggers builds whenever code is pushed to GitHub and deploys the application to an AWS EC2 instance.

## Technologies Used

* Jenkins
* GitHub
* GitHub Webhooks
* AWS EC2
* AWS IAM
* Linux (Ubuntu)
* Python
* SSH

## Workflow

1. Developer pushes code to GitHub
2. GitHub Webhook triggers Jenkins
3. Jenkins executes build and test stages
4. Jenkins connects to AWS EC2 using SSH
5. Application is deployed automatically

## Features

* Automated CI/CD pipeline
* GitHub Webhook integration
* Secure SSH deployment
* AWS EC2 hosting
* Linux server administration
